스타일 문법과 기본 선택자

CSS(Cascade Style Sheet) - 웹 문서에 종속된 스타일(서식)

문법 : 선택자 { 속성:값; 속성:값 }

예시) header nav a { color:red; font-weight:900; }

선택자(Selector) 란?

스타일(서식)이 부여될 대상

스타일 정의하고, 지정하는 방법

내부 스타일 : 현재 문서에 스타일을 적용하는 경우이며, style태그를 활용

외부 스타일 : 별도의 스타일(*.css)을 지정하여 link태그로 불러와 사용하는 경우

인라인 스타일 : 해당 태그에 원하는 스타일을 style 속성으로 지정하는 경우
